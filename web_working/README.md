# FD SMS

## Dev Setup

### Setup Python

```sh
# Create venv
$ python -m venv .venv/
# Activate source
$ source .venv/bin/activate
# Install requirements
$ pip install -r requirements.txt
```

### Setup Node

```sh
npm i
```

### Basic Run Commands

```sh
# Start vite server
$ npm run dev
# Start python server
$ npm run server
# Starts python server and vite server
$ npm start
```
